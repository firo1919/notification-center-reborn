# Notification Center Reborn

**Version 24.02 — Modernized for GNOME 45, 46, 47, and 50+**

A GNOME Shell extension for detaching the notification center and highly customizing your top panel.  
_This is a modernized ESM-compliant fork of the original Notification Center extension, updated with clean Adwaita preferences._

## Installation

### From GitHub

Download or clone this repository.

Open Terminal in the extracted folder and run:

```bash
chmod +x INSTALL.sh
./INSTALL.sh
```

Restart GNOME Shell:

- **Xorg**: Press `Alt` + `F2`, type `r`, and press `Enter`.
- **Wayland**: Log out and log back in.

Enable the extension using the **Extensions** application.
